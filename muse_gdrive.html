muse_gdrive package¶

Module contents¶

Module allowing simple manipulations of files on Google Drive.

muse_gdrive.connect(secret_file_path, email_address)¶

Authenticates to Google Drive for a given user’s email address.

muse_gdrive.delete(google_drive, path)¶

Deletes a file or folder (and children) on Google Drive.

muse_gdrive.download(google_drive, source, dest)¶

Copies a file from Google Drive to the local filesystem.

muse_gdrive.get_id_by_name(google_drive, name, parent_id)¶

Gets the ID of a file or folder contained in a given parent folder by name.

muse_gdrive.get_id_by_path(google_drive, path)¶

Gets the ID of a file or folder by its absolute path.

muse_gdrive.list_files(google_drive, path)¶

Lists the contents of a folder on Google Drive.

muse_gdrive.make_directory(google_drive, path)¶

Create a directory at the given location.

muse_gdrive.upload(google_drive, source, dest)¶

Copies a file from the local filesystem to Google Drive.
